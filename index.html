Exemplos de Mapas

Como criar um mapa e marcadores nele:

Em um arquivo html,com a biblioteca Leaflet já adicionada, você deve abrir um script e nele colocar: 

                var  map  = L.map('map').setView([0, 0], 1);
            
                L.tileLayer('https://api.maptiler.com/maps/streets-v2/{z}/{x}/{y}.png?key=0xLQbt4WJeeuwW3p39Hy', {
                attribution: '© MapTiler © OpenStreetMap contributors'
                }).addTo(map);
                var  marker = L.marker([51.5, -0.09]).addTo(map);

Como adicionar marcações como circulos e triângulos ao meu mapa?

No arquivo com um mapa, voce pode colocar os seguintes poligonos 

                var circle  = L.circle([51.508, -0.11], {
                    color: 'red',
                    fillColor: '#f03',
                    fillOpacity: 0.5,
                    radius: 1500
                }).addTo(map);
            
                var polygon  = L.polygon([
                    [51.509, -0.08],
                    [51.503, -0.06],
                    [51.51, -0.047]
                ]).addTo(map);

Como utilizar outros estilos de Mapas?

Para começar, você pode encontrar outros provedores de mapas no site Leaflet Providers Preview. Uma vez escolhido o provedor de mapas desejado, copie o código fornecido pelo provedor e cole-o em seu projeto. 

                 var OpenStreetMap_HOT = L.tileLayer('https://{s}.tile.openstreetmap.fr/hot/{z}/{x}/{y}.png', {
                    maxZoom: 19,
                    attribution: '© OpenStreetMap contributors, Tiles style by Humanitarian OpenStreetMap Team hosted by OpenStreetMap France'
                }).addTo(map);

Como colocar escala? Ou como colocar Marca D'agua

Para adicionar a escala e uma marca d'água ao seu mapa, siga estas etapas simples:

                var L.control.scale({
                    metric: true,
                    imperial: false,
                    position: 'topright'
                }).addTo(map);
                
                L.Control.Watermark = L.Control.extend({
                    onAdd: function(map) {
                        var img = L.DomUtil.create('img');
                        img.src = 'logo.png';
                        img.style.width = '75px';
                        return img;
                    },
                    onRemove: function(map) {}
                });
                
                L.control.watermark = function(opts) {
                    return new L.Control.Watermark(opts);
                }
                
                L.control.watermark({ position: 'bottomleft' }).addTo(map);

Para colocar a barra de pesquisa é simples, apenas um passo é necessitado:

                L.Control.geocoder().addTo(map);

Como colocar um marcador que engloba outros marcadores e me mostra quantos tem em determinada regiâo

Para realizar este feito, voce deve baixar a pasta de MarkerCluster disponibilizada neste link: 'https://github.com/Leaflet/Leaflet.markercluster' apos baixar, voce pega a pasta dist e coloque em sua pasta do projeto, importando em seu projeto os scripts e links necessários para o funcionamento(Este agrupamento de marcadores é comumento é usado quando há muito marcadores devido a algum arquivo sendo json ou csv, está sendo utilizado uma importação de um arquivo json para criar estes marcadores, algo que já foi ensinado acima). Após disto terá que ser feito as seguintes comandas: 

                < link rel="stylesheet" href="./dist/MarkerCluster.css">   /Importações
                < link rel="stylesheet" href="./dist/MarkerCluster.Default.css"> /Importações
                < script src="Data\data.js"> /Importações
                < script src="./dist/leaflet.markercluster.js"> /Importações

                const geojsonmarkerOptions = {
                    radius: 8,
                    fillcolor: "#ff7800",
                    color: "#000",
                    weight: 1,
                    opacity: 1,
                    fillOpacity: 0.8
                };
                
                const markers = L.markerClusterGroup();
                L.geoJSON(data, {
                    pointToLayer: function (feature, latlgn) {
                        return markers.addLayer(L.circleMarker(latlgn, geojsonmarkerOptions));
                    }
                }).addTo(map);
                
                map.addLayer(markers);
